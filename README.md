# Iraq Air Quality Monitoring Radar (Sentinel-5P)

 Project Overview
This tool is designed to monitor and analyze air pollution across Iraq using **Google Earth Engine (GEE)** and **Sentinel-5P** satellite data. It provides an automated environmental assessment for researchers and decision-makers.

---

 Monitored Pollutants
The system analyzes the concentration of critical gases:
* **NO2 (Nitrogen Dioxide):** Monitoring industrial and urban traffic emissions.
* **SO2 (Sulphur Dioxide):** Detecting emissions from power plants and oil refineries.
* **CO & CO2:** Analyzing combustion levels and greenhouse gas trends.
* **CH4 (Methane):** Tracking environmental leaks and agricultural impact.

 How to Use
1. Copy the source code from the `src` folder in this repository.
2. Open [Google Earth Engine Code Editor](https://code.earthengine.google.com/).
3. Paste the code and use the **Polygon Tool** to select your study area.
4. Click **"Run Analysis"** to generate the real-time report.
